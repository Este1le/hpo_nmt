Evaluation

The problem is a multiobjective hyperparameter optimization problem. We want to find the best hyperparameter settings that lead to good accuracy-speed tradeoff on a given dataset.

This sample competition allows you to submit:

  • Only prediction results (no code).
  • Code that includes logic or pre-trained model for automatically sampling hyperparameter configurations

The submissions are evaluated using the number of pareto solutions found after a fixed budget ("fb" metric).
